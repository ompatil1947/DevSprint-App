# 🤖 RoboDate — Swipe Right on Circuits ❤️

Welcome to **RoboDate**, the most *electrifyingly awkward* dating app you’ll ever use.
Forget humans. Forget compatibility algorithms.
Here, you date **robots with terrible dad jokes** 🤖💀

---

## 🚀 Overview

**RoboDate** is a fun, procedurally generated dating app where every profile is a **unique robot** with:

* 🧠 A generated name
* 🎂 A (slightly suspicious) age
* 😂 A painfully bad dad joke pickup line

You swipe:

* 👉 **Like** (maybe love sparks… or sparks literally fly ⚡)
* 👈 **Pass** (ghosting is socially acceptable here)

And the universe decides your fate.

---

## ✨ Features

### 🔁 Procedural Robot Generation

* No two robots are the same
* Randomly generated:

  * Names (e.g., *RX-9000, Bytey McBot, Sir Circuits*)
  * Ages (from freshly assembled to ancient relics)
  * Pickup lines (the worse, the better)

---

### 😂 Dad Joke Engine

Every robot comes with a pickup line like:

> "Are you WiFi? Because I'm feeling a connection... but it's unstable."

Expect:

* Cringe ✔️
* Laugh ✔️
* Existential regret ✔️

---

### 💘 Swipe Mechanics

* **Like ❤️** → You’re interested
* **Pass ❌** → Not your type (maybe too many bolts)

---

### 🌌 Match System

* Matching is NOT guaranteed
* A mysterious "universe logic" decides:

  * Match 💖
  * Rejection 💔
  * Or something… unexpected 👀

---

### 🎨 Beautiful UI (Optional Enhancement)

* Smooth swipe animations
* Neon cyberpunk vibes
* Glowing robot cards

---

## 🧠 How It Works

1. App generates a robot profile
2. Displays:

   * Name
   * Age
   * Dad joke pickup line
3. User swipes Like/Pass
4. System randomly determines match outcome
5. Repeat → Endless robotic romance ♾️
